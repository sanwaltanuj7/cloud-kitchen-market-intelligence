#  Cloud Kitchen Market Intelligence Project

## 📌 Overview

This project presents a market intelligence analysis of food delivery restaurants in Nashik using data collected from Zomato. The study focuses on identifying cuisine trends, market saturation, cloud kitchen opportunities, and operational insights through data collection, data cleaning, SQL analysis, and business intelligence techniques.

---

## 🎯 Project Objectives

- Collect and organize restaurant data from a selected locality in Nashik.
- Investigate API and network requests using browser developer tools.
- Clean and standardize real-world restaurant datasets.
- Perform SQL-based business analysis.
- Generate actionable insights for launching a cloud kitchen business.
