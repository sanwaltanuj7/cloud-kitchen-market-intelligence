# cloud-kitchen-market-intelligence
Market Intelligence Study of Food Delivery Restaurants in Nashik using Zomato/Swiggy data, SQL analysis, API investigation, and data cleaning.

##Overviiew

This project analyzes restaurant data from Nashik to identify cuisine trends, market saturation, and cloud kitchen opportunities.

- Data Collection
- API Investigation
- Data Cleaning
- SQL Analysis
- Business Insights

## Tools Used

- Python
- Pandas
- Google Colab
- SQL
- Git & GitHub
- Chrome Developer Tools

- Raw Dataset
- Cleaned Dataset
- Menu Dataset
- SQL Queries

- Most saturated cuisines
- Underrepresented cuisines
